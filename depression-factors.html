Depression Factors Classification

View Project

Keywords: Machine Learning, Data Mining, Java WekaAPI, Scikit-Learn, GridSearchCV, Hyper-parameter tuning, SVM, Logistic Regression, OneR, Naive Bayes, J48, IBk, AdaBoost, Random Forest, Extra Tree, Depression factors, Depression levels, Prediction model, CVParameterSelection.

From March to May 2024, I worked on a Machine Learning project titled "Depression Factors Classification" as part of a Data Mining course. This project aimed to develop a prediction model to analyze the relationship between various depression factors and depression levels.

My contributions primarily involved base-model implementation and hyper-parameter tuning. We employed the Java WekaAPI to construct our models, focusing on evaluating six machine learning algorithms: Support Vector Machine (SVM), Logistic Regression, OneR, Naive Bayes, J48, and IBk. Additionally, we explored the performance of three ensemble methods: AdaBoost, Random Forest, and Extra Tree.

To optimize the models, we utilized CVParameterSelection from Java WekaAPI and GridSearchCV from Scikit-learn for hyper-parameter tuning. This systematic approach ensured that we identified the best parameters for each model, enhancing their predictive accuracy and overall performance. The project's rigorous evaluation process allowed us to comprehensively understand how different algorithms and tuning techniques impacted the classification of depression factors.
