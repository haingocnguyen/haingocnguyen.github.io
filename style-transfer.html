View Project

Keywords: Style Transfer, Image Processing, Optimal Transport, VGG16, PyTorch, Machine Learning, Feature Extraction, Gradient Descent, Algorithm Development, Mathematical Research

In December 2023 and January 2024, I worked on an Image Processing project titled "Style Transfer" for the Optimization and Applications course. My role involved contributing to algorithm development and conducting mathematical research tasks.

The project focused on using machine learning and optimal transport algorithms for style transfer in images. We aimed to blend features from style and content images to create aesthetically pleasing outputs. For feature extraction, we utilized the VGG16 neural network to capture essential details from both style and content images.

The implementation was carried out using PyTorch, where we developed and optimized machine learning algorithms with gradient descent. This approach allowed us to effectively transfer stylistic elements from one image to another, showcasing the practical applications of advanced optimization techniques in image processing.
