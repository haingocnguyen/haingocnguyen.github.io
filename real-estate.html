Real Estate Crawler

View Project

Keywords: Real Estate, Web Crawling, Selenium, Python, Data Collection, House Prices, Dynamic Elements, Property Descriptions, CSV, XLSX, Database

In my ongoing project, "Real Estate Crawler," which is part of a larger scientific research initiative, I have been working since March 2024 to develop a program that crawls real-time real estate data using Python and Selenium. My contributions focus on implementing the crawling mechanism to gather key features that affect house prices.

The project aims to automate the process of navigating real estate websites, interacting with dynamic elements, and collecting essential data such as property descriptions, prices, locations, and images. Using Selenium, we ensure that our crawler can handle complex web elements and retrieve accurate, up-to-date information.

The collected data is stored in structured formats like CSV and XLSX files, while images are saved in a designated database. This organized storage facilitates further analysis and visualization, contributing valuable insights to our research on real estate market trends. This project showcases the effective use of web crawling techniques to gather and manage large datasets from dynamic online sources.
